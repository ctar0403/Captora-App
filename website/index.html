How To Use

Three steps to instant OCR

From zero to copied text in under two seconds.

1

Launch & Setup

Install Captora and optionally add Tesseract language packs. The app sits silently in your system tray, always ready when you need it.

2

Select Area (Win + Q)

Press Win + Q at any time. Your cursor becomes a crosshair — drag to draw a rectangle around the text you want to capture.

3

Instant Result

The recognized text is copied to your clipboard and shown in an optional popup window. Paste it anywhere immediately.

Optimization

Get better accuracy

Three quick settings that dramatically improve OCR results.

🔍

OCR Scale Factor

Increase the scale factor in Settings → OCR to upscale small or low-resolution text before recognition. A value of 2× or 3× often improves accuracy for small fonts dramatically.

📝

Whitelist Characters

Capturing numbers-only content like prices or codes? Set a character whitelist to 0123456789 to eliminate misreads and speed up recognition.

📐

Deskewing Images

Enable the deskew option when capturing rotated or tilted text. Tesseract performs significantly better on straight, horizontal text lines.

Troubleshooting

Common issues & fixes

Quick answers to the most frequently reported problems.

📂

Tesseract data path not found

Go to Settings → Tesseract and verify the path points to your tessdata folder. The default path is C:\Program Files\Captora\tessdata. Re-run the installer if the folder is missing.

⚙️

Application won't start or crashes on launch

Install the latest Microsoft Visual C++ Redistributable (x64) from the official Microsoft website. Missing this runtime causes an immediate silent crash on many systems.

🖥️

Capture box is misaligned on high-DPI screens

Right-click Captora.exe → Properties → Compatibility → Change high DPI settings → Override the scaling behavior, set it to Application. Apply and restart the app.

Pro Tips

Work smarter, not harder

Power-user tricks to squeeze more speed out of Captora.

01

Clipboard-Only Mode

Disable the result popup in Settings to make captures completely silent — text appears in your clipboard with zero interruption.

02

Use Character Whitelists

For domain-specific captures like prices, codes, or dates, whitelists eliminate false positives and cut processing time significantly.

03

Hotkey Language Switching

Assign a hotkey to each language you use frequently. Switch between English and Japanese OCR in milliseconds without touching the menu.

04

Contrast & Preprocessing

Enable preprocessing filters (grayscale + contrast boost) for difficult text on complex backgrounds to help Tesseract isolate characters cleanly.
